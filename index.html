Le Wagon UI components

Getting started


Before you use one of our components:

  • Make sure you understand the HTML structure of this component.
  • Click on the "View CSS code" button and make sure you understand the associated CSS code and its "graphical logic".
  • Add a new .SCSS partial file (e.g. _avatar.scss) in your stylesheets components folder, copy/paste our CSS code inside, and then use the HTML code wherever you want.
